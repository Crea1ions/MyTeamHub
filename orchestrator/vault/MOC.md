## 🧠 1. COMMENT LE SYSTÈME FONCTIONNE

L'Orchestrator est :

> 🧠 **un système de cognition structuré basé sur fichiers**

---

### 🧱 Les 4 phases du système

```text
🎨 studio       → brainstorm (capture d'idées)
🧪 agent-dev    → pensée (court terme)
📦 projects     → structure (moyen terme)
📚 knowledge    → vérité (long terme)
```

### 🔁 Cycle réel

```
idée brute → exploration → décision → structuration → capitalisation
```

### ❗ Principe clé

> Rien n'est automatique  
> Tout est explicite
